BALI YOGA | UX/UI _ Desktop, Mobile, App

BaliYoga is a yoga center based in Milan which offers yoga classes with various styles and levels, and this is a Case Study.

Services and advantages:

  • 2 types of lessons: in studio or interactive
  • Professional training courses
  • Corporate classes
  • Workshop
  • 25 teachers
  • 14 yoga styles
  • 3 levels: basic, intermediate, advanced

Logo redesign

The "Thasadith - Bold" font takes up the elongated shapes of the petal:

Problem Statement

The website presents 3 main problems that cause usability issues:

  1. from the homepage is not clear the main service offered;
  2. the structure of the sitemap is confusing;
  3. too many CTAs that prevent a smooth user flow.
Assumptions:

  • Change sitemap and IA
  • Build linear user flows
  • Move some CTAs

Usability test

I assumed to run 3 tests with various tasks on the main aspects of the interfaces to determine the level of usability.

  • In-person, unmoderated
  • Time assigned
  • Target of 5 people (female and male) with medium/high devices proficiency (because online bookingis always requested)



Criteria

  • Task completion rate
  • Time to completion
  • Last click
  • Task duration
  • Average time on page and number of clicks
  • Error rate

TEST 1 Desktop & Mobile Interface

Method
Scripted, unmoderated test.


Scenario A
You have never done yoga but you are interested in practicing it. You have been recommended the BaliYoga center which offers 1 free trial lesson.

Closed task A
You are on the homepage of the website and you want to request / book a free trial lesson. Where do you click?

Goals A
Understanding if the user who interacts with the site for the first time immediately learns that he can get a free lesson. Especially in the mobile version where the path is only possible from the home.



Scenario B
You want to purchase a specific package of yoga classes to follow at your home.

Open-ended task B
You must purchase a package of "5 Lessons" only interactive. What path do you take?

Goals B
It is possible to proceed in various ways, by:

  • Hero in the home
  • Menu > Lessons

The aim is to understand if the splitting of gym / interactive lessons packages is clear enough and if the user quickly understands that some packages are mixed.

Scenario C
You already have a subscription to yoga courses and you just want to book your next lesson on the website's Calendar, because you know that booking is mandatory for each lesson before taking it.

Open-ended task C
Book the evening lesson on Friday 19 March held by teacher Carlotta.


Goals C
It is possible to complete the task by:
  • CTA on page Lessons in the gym
  • Menu > Lessons > Calendar
  • Hyperlink Calendar in Prices

Understand if the booking procedure is fluid.

TEST 2 Desktop & Mobile Interface

Method
5 second test.


Scenario A
Gym Lessons page is shown and the user is asked to memorize as many elements as possible within 5 seconds. When the time is up, some questions are asked.


Questions A

  • What is the service offered in the page you saw?
  • What can you do through this page?
  • If you click the green button what happens?
  • What can you do if you click the blue button?
  • Does this page give you information on interactive lessons?


Goals A
Find out if:

  • UI communicates the message correctly
  • Contents are immediately clear
  • CTAs are perceived correctly

TEST 3 App Interface

Method
Scripted, unmoderated test.


Scenario A
You are in a rush and want to view specific courses through filters.


Closed task A
Set filters to view only the advanced courses held on the afternoon of Friday, March 19.

Goals A
Find out if:

  • Users accesses the features intuitively.
  • Contents are immediately clear
  • The flow is clean and linear



Scenario B
You have tired eyesight but you are curious to explore the app and its lesson calendar.

Closed task B
View contents by setting night mode.

Goals B
Understand if the user is able to set the required mode immediately.



Scenario C
You have just installed the BaliYoga App and haven't explored it yet, but you know their site.

Questions C

  • In your opinion, what is inside the Bali Yoga icon?
  • What content do you expect to find if you click it?

Goal C
Understand if the categories provided are sufficient or if on the contrary they are superfluous.
